# Pomodoro Timer ⏱️

A customizable Pomodoro Timer designed to improve focus and productivity.  
The application includes persistent settings, detailed study statistics, and quality-of-life features like dark mode and sound alerts.

## Features

- Start, pause, and reset the timer
- Focus and break modes
- Customizable focus and break durations
- Sound notification when a session ends
- Dark mode toggle
- Persistent settings (saved between sessions)

### Study Statistics

- Daily study statistics
- Weekly study statistics
- Monthly study statistics
- All statistics persist across page reloads and browser restarts

## Tech Stack

- HTML
- CSS
- JavaScript
- Local Storage (for data persistence)

## How It Works

- Timer logic controls focus and break cycles
- User preferences (minutes, theme) are stored locally
- Study sessions are automatically recorded
- Daily, Weekly and monthly stats are calculated from stored data
- No backend required
